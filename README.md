# CarHop Rentals

CarHop Rentals is a modern car rental web application that allows users to browse vehicles and explore rental options through a clean and responsive interface.

This project focuses mainly on frontend development and demonstrates building a real-world style application using React and TypeScript. The backend and database will be powered by **Supabase**.

---

## Tech Stack

**Frontend**

- React
- TypeScript
- Tailwind CSS
- Vite

**Backend (Planned)**

- Supabase
- PostgreSQL

---

## Features

- Browse available rental cars
- View vehicle details
- Responsive design for mobile and desktop
- Component-based UI architecture

**Planned Features**

- User authentication
- Car booking system
- Reservation management
- Admin dashboard for managing cars and bookings

> 🚧 This project is currently a work in progress.
